A twitter bot that will tweet about the weather. It may either tweet something related to the current weather as determined by Yahoo or will tweet what the condition is. 
Written in Python using Python Twitter and Pywapi. 
Runs using Cron on a friend's server and posts to @rationalWeather every 3 hours. 

Things I would like to add before winter: 
  Specific tweets if the wind chill gets below a certain level.
